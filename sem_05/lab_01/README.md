Изучить основы построения микропроцессорных систем на ПЛИС.
Ознакомиться с принципами построения систем на кристалле (СНК) на основе ПЛИС.

Выполнить проектирование и верификацию системы с использованием отладочного комплекта Altera DE1Board.
